# 关于本站

本网站使用 **Astro** 框架构建，采用 [Mizuki](https://github.com/matsuzaka-yuki/mizuki) 主题。

::github{repo="matsuzaka-yuki/Mizuki"}

## 🌟 主题特色

### 🎨 设计与用户体验
- **现代优雅设计** - 简洁、极简主义的界面与优美的排版
- **完全响应式** - 为从移动设备到桌面的所有设备优化
- **深色/浅色模式** - 自动主题切换，带有平滑过渡
- **优美排版** - 使用 JetBrains Mono 字体增强可读性
- **流畅动画** - 流畅的页面过渡和交互元素

### 🔍 内容与搜索
- **高级搜索** - 由 [Pagefind](https://pagefind.app/) 提供支持，快速准确的结果
- **增强 Markdown** - 扩展语法，支持代码高亮和数学公式
- **交互式目录** - 长文章的自动滚动导航
- **RSS 订阅生成** - 通过自动订阅生成保持更新
- **阅读时间估算** - 了解文章需要多长时间阅读
- **文章分类** - 使用标签和分类组织内容

### 📱 特殊页面
- **动漫追踪页面** - 通过评分追踪您的动漫观看进度
- **友链页面** - 使用精美卡片展示友链网站
- **日记/动态页面** - 像社交媒体帖子一样分享生活瞬间
- **归档页面** - 所有文章的组织时间线视图
- **关于页面** - 可自定义的个人介绍（就是此页面！）

### 🛠 技术特色
- **增强代码块** - 由 [Expressive Code](https://expressive-code.com/) 提供支持
- **数学公式支持** - 使用 KaTeX 渲染 LaTeX 数学表达式
- **图片优化** - 带有懒加载的 PhotoSwipe 画廊
- **SEO 优化** - 内置站点地图和元标签，提升搜索可见性
- **性能优化** - 通过缓存和优化实现快速加载
- **评论系统就绪** - 支持 Twikoo 评论集成

### 🎯 高级 Markdown 功能
- **标注与提示** - 使用 `> [!NOTE]`、`> [!TIP]`、`> [!WARNING]` 的优美信息框
- **数学公式** - 使用 `$行内$` 和 `$$块级$$` 语法编写 LaTeX 数学公式
- **GitHub 卡片** - 使用 `::github{repo="用户/仓库"}` 嵌入仓库卡片
- **语法高亮** - 带行号的高级代码高亮
- **复制代码按钮** - 便捷的代码复制功能

*使用 Astro 构建，❤️ 灵感来源于现代网页设计原则。*
